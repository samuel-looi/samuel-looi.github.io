CSS Positions: Relative absolute fixed

The position CSS attribute, changes how the HTML element is positioned. Here I will be explaining the following position values: relative, absolute, fixed

Relative

The relative value does not change the position of the element it is assigned to, unless the positioning attributes (top, right, bottom, left) are used.

The value assigned to the positioning attributes (top, right, bottom, left) will cause the element to be moved away by the amount specified from the direction (top, right, bottom, left) mentioned.

With the following style code:

position:relative; top:20px;

The element it is assigned to will be moved 20px away from the top from its previous (original) position. In other words, it will move down 20px.

Absolute

The element assigned with this value will be positioned absolutely from either the closest that parent has the position value of relative, absolute or fixed. Otherwise the element is position relative to the HTML element

When this value is used, the element is taken out of the DOM flow, and thus other elements will act as if it's not there.

Note that this element will still be in its original position until the positioning attributes (top, right, bottom, left) as used.

Fixed

The CSS position fixed is the exact same as the CSS position absolute, except the top, right, bottom, left positioning attributes refering to the viewport.

With the following styling code:

position:fixed; top:5px;

The element will be positioned in relation to the page, so the element will be moved 5 pixels from the top of the page.
