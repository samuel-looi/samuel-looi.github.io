An analogy to describe the differences between HTML and CSS

One could describe HTML and CSS to a home. A home has a general layout, has rooms of certain sizes, furniture and accessories, all of which represent the HTML - they are the content of the home. The CSS would be the looks of the content of the home, such as how many rooms there are, what their sizes are and the type of furniture and accessories.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'. (But not those ones).

The basic control flow of JS is that code is executed top to bottom. In for loops, the code is executed repeatedly until a certain condition is no longer satisfied. A real life example of this could be flossing your teeth. You may want to floss your teeth, so you floss each tooth one by one until all teeth have been flossed (the condition), much like how an array would iterate over each element one by one executing a piece of code until all have been iterated (the condition).

Functions are just a sequence of commands, that can be called once declared. It can be useful to prevent rewriting the same code again and again, which increases legibility, productivity and maintainability.
